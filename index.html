🎯 O Que é Nexiflow Approvals?

Nexiflow é um gerenciador inteligente de projetos imobiliários que automatiza a avaliação, aprovação e análise de viabilidade de projetos.

Ele funciona como um "gerente de projetos" que:

  • ✅ Recebe suas informações via formulário online seguro
  • ✅ Avalia viabilidade usando dados do IBGE e análise especializada
  • ✅ Aprova ou Rejeita de forma estruturada e transparente
  • ✅ Guia você através das próximas etapas com suporte completo
🚀 As 4 Fases do Seu Projeto
▼
📋 Fase 1: Envio Inicial
Você preenche um formulário detalhado com informações do seu projeto. O sistema valida automaticamente todos os campos, calcula um score de viabilidade e gera um Número de Protocolo único (exemplo: NXM-2026-ABC12345) para rastreamento. Você recebe confirmação por email.
🔍 Fase 2: Análise do Revisor
Um especialista em projetos imobiliários analisa seu projeto em detalhes. Ele pode:
  • ✓ Aprovar - Seu projeto é viável!
  • ⚠ Pedir Correções - Alguns campos precisam ajuste
  • ✗ Rejeitar - Projeto não se enquadra (com motivo claro)
📝 Fase 3: Próximas Etapas
Se Lançamento/Remanescente: Vai direto para a proposta (Fase 4)
Se Análise de Áreas: Você preenche um formulário complementar com mais informações técnicas
💰 Fase 4: Proposta Customizada
Você recebe uma análise completa e customizada com:
  • ✓ Análise de mercado local
  • ✓ Estimativa de viabilidade econômica
  • ✓ Recomendações de posicionamento
  • ✓ Próximos passos sugeridos
Enviamos um PDF seguro via link único.
📌 Exemplo Real: O Projeto do João
▼

João quer lançar um novo loteamento em São Paulo e precisa validar a viabilidade.

Fase 1: Preenchimento do Formulário

João acessa o site e escolhe:

  • Tipo: Lançamento
  • Subtipo: Loteamento
Categoria 1: Informações Gerais
  • Nome do negócio: "Loteamento Vila Nova"
  • Tipo de Estudo: "Parcelamento de Solo"
  • Email do proprietário: contato@vilanova.com.br
Categoria 2: Localização
  • Estado: São Paulo
  • Cidade: Osasco
  • CEP da área: 06085-100
Categoria 3: Dados Demográficos
  • População: 720.000
  • Público pendular (por mês): 150.000
  • Público flutuante: 80.000
  • PIB per capita: R$ 45.000
  • IDH: 0,789
  • Atividade principal: Comércio e Serviços
Categoria 4: Características do Empreendimento
  • Tipologia: Casa
  • Destinação: 1ª moradia
  • Status da obra: Terreno limpo, pronto
  • Metragens: Lote 300m² a 500m²
  • Total de unidades: 80 lotes
  • VGV: R$ 8.000.000
Categoria 5-9: Dados de Mercado, Construtora, Negociação, Documentos

João preenche com detalhes sobre preço médio, lançamentos concorrentes, histórico da construtora, documentação anexada, etc.

✅ O que acontece depois?
  1. Sistema valida todos os campos obrigatórios
  2. Sistema calcula um SCORE usando dados do IBGE (população, PIB)
  3. Gera número: NXM-2026-ABC12345
  4. Email enviado: "Seu projeto foi registrado. Protocolo: NXM-2026-ABC12345"
Fase 2: Revisor Analisa

Um especialista vê o projeto de João e pode:

  • ✅ APROVAR: "Seu projeto é viável! Vamos adiante." → Email com acesso
  • ❌ REJEITAR: "Lamentamos, mercado saturado" → Email com motivo
  • 🔄 PEDIR CORREÇÕES: "Alguns campos precisam ajuste" → Email com link para corrigir
Fase 3-4: Próximas Etapas

Como João escolheu "Lançamento", ele vai direto para a proposta (não há Fase 3).

O revisor prepara uma análise completa com recomendações de mercado e envia um PDF seguro.

📍 Como Rastrear Seu Projeto
▼

A qualquer momento, você pode acompanhar o status do seu projeto:

Opção A: Via Protocolo
  1. Acesse a plataforma
  2. Clique em "Rastrear Projeto"
  3. Digite seu protocolo: NXM-2026-ABC12345
  4. Clique em "Buscar"

Você verá:

  • Status atual (em análise / aprovado / rejeitado / aguardando correções)
  • Data de envio
  • Timeline completa de eventos
Opção B: Via Email
  1. Abra o email que recebeu
  2. Clique no link de rastreamento
  3. Você é levado direto ao seu projeto
💡 Protocolo é Público

Você pode compartilhar seu protocolo com qualquer um - ele apenas mostra o status. Para acessar o formulário ou proposta, é preciso do link enviado por email (seguro).

👤 Guia Prático de Uso
1️⃣ Enviar um Novo Projeto
▼
Passo 1: Acessar a Plataforma
  • Abra: http://localhost:8081
  • Clique em "Novo Projeto" ou "Enviar Análise"
Passo 2: Escolher o Tipo de Projeto

Você vai ver 4 opções:

Tipo Quando Usar Próxima Etapa
Análise de Áreas Você tem um terreno bruto e quer saber se é viável construir Fase 3 complementar + Proposta
Lançamento Você está lançando um novo imóvel/condomínio no mercado Direto para Proposta
Remanescente Seu projeto antigo ainda tem unidades para vender Direto para Proposta
Análise de Áreas (Incorporador) Você é incorporador com terreno disponível Fase 3 complementar + Proposta
Passo 3: Preencher o Formulário

O formulário é dividido em categorias (abas).

💡 Dicas de Navegação
  • Clique "Próximo" (seta direita) para avançar
  • Clique "Anterior" (seta esquerda) para voltar
  • Clique no nome da categoria para pular direto
⚠️ Campos Obrigatórios
  • ⭐ Campos com asterisco (*) são obrigatórios
  • 💡 Campos com "?" têm ajuda extra ao passar mouse
  • 🔴 Se ficar vermelho, revise antes de continuar
Passo 4: Dados do Responsável

Na primeira tela, preencha:

  • Seu Nome Completo
  • Seu Email
  • Seu Telefone

Esses dados aparecem em todos os emails enviados.

Passo 5: Submeter

Clique em "Enviar Projeto" no final.

O sistema vai:

  1. Validar todos os campos
  2. Calcular o score
  3. Gerar protocolo
  4. Enviar email de confirmação

Espere a animação terminar. ⏳

2️⃣ Rastrear Seu Projeto
▼
Opção A: Via Protocolo
  1. Acesse a plataforma
  2. Clique em "Rastrear Projeto"
  3. Copie/cole seu protocolo: NXM-2026-ABC12345
  4. Clique em "Buscar"

Você verá:

  • Status atual (em análise / aprovado / rejeitado)
  • Data de envio
  • Timeline de eventos
Opção B: Via Email
  1. Abra o email que recebeu
  2. Clique no link de rastreamento
  3. Você é levado direto ao seu projeto
3️⃣ Se Seu Projeto Precisa de Correções
▼

Se o revisor respondeu "Aguardando Correções":

  1. Email recebido com link de acesso
  2. Clique no link → você volta ao formulário
  3. Campos vermelhos = o que precisa corrigir
  4. Sistema navega automaticamente para o 1º erro
  5. Corrija os campos indicados
  6. Clique "Reenviar"
✅ O que acontece?

O sistema apaga a rejeição anterior e trata como novo envio. Você recebe um novo protocolo e o revisor analisa novamente.

4️⃣ Se Seu Projeto Foi Aprovado
▼

Você receberá email dizendo: "Seu projeto foi aprovado!" 📧

SE é Lançamento/Remanescente:
  • Aguarde a proposta estar pronta
  • Email com link para baixar proposta PDF
  • Sem Fase 3 complementar
SE é Análise de Áreas:
  • Email com link para formulário complementar
  • Isso é a FASE 3 (tem mais ~35 perguntas)
  • Preenchida a Fase 3, aí sim vem a proposta
5️⃣ Acessar a Proposta
▼

Você recebeu email com:

Assunto: Sua Proposta Está Pronta! Link de acesso: http://localhost:8081/proposta/123abc?token=xyz789

Clique no link (vale copiar/colar também):

  • ✓ Sem necessidade de login
  • ✓ Link é único e seguro
  • ✓ Abre em PDF no navegador
  • ✓ Pode baixar/imprimir

⚠️ Score é Confidencial
  • Score é calculado durante Phase 1

🔐 Segurança: Tipos de Token
▼

✅ Regras de Acesso
  • Cliente vê APENAS seu projeto (via token/email)
  • Reviewer vê TODOS (com JWT autenticado)

  • Ninguém vê score sem ?reviewer=true
  • Proposta = APENAS email/token, nunca login

💡 Método de Entrega

Emails são enviados via SMTP direto do backend (não via webhooks N8N). Cada evento dispara um template HTML e conecta ao provider de email configurado.

❓ Perguntas Frequentes (FAQ)
❓ Como o scoring funciona exatamente?
▼

O sistema usa população + PIB per capita do IBGE + dados que você preenche (topografia, acessos, infraestrutura, mercado) para gerar uma pontuação de 0-100.

Isso ajuda o revisor a ter uma primeira impressão objetiva do projeto. Score alto ≠ garantia de aprovação (revisor tem palavra final).

❓ Posso editar meu projeto depois de enviar?
▼

Apenas se o revisor pedir correções.

Caso contrário, você precisa enviar um novo projeto (vai gerar novo protocolo). O sistema trata reenvios como novos projetos para auditoria.

❓ Por quanto tempo meu link de acesso funciona?
▼

O link nunca expira naturalmente.

Um admin pode revogá-lo manualmente se necessário. Você pode usar o link quantas vezes quiser, mesmo meses depois.

❓ O revisor vê meus comentários (observações)?
▼

Sim, tudo é registrado no sistema.

É um "diário de bordo" completo do projeto. Seu campo "Observações" fica visível no dashboard do revisor.

❓ Posso compartilhar meu protocolo com alguém?
▼

Sim! Protocolo é público.

Qualquer um pode rastrear o status usando o protocolo. Mas o acesso ao formulário/proposta requer o link enviado por email (seguro, token-based).

❓ E se o revisor não responder?
▼

Depende da SLA (Acordo de Nível de Serviço) do seu contrato.

O sistema registra tudo (data/hora de envio, análise, resposta) para auditoria. Você pode contatar o suporte se exceder o prazo acordado.

❓ Quantas vezes posso corrigir antes de ser rejeitado?
▼

Quantas vezes for necessário.

Não há limite de correções. O sistema trata cada reenvio como novo envio. O revisor pode optar por rejeitar definitivamente se achar apropriado, mas não há regra de limite automático.

❓ Meu projeto foi rejeitado. Posso reenviar?
▼

Sim, como novo projeto.

Uma rejeição é permanente naquele protocolo. Você pode enviar um novo projeto com ajustes. Vai gerar novo protocolo e será analisado novamente do zero.

❓ Phase 3 é obrigatória para todos?
▼

Não. Apenas para Análise de Áreas.

Se você enviou Lançamento ou Remanescente, vai direto para a Proposta (Phase 4) após aprovação. Phase 3 só existe para validar dados complementares de terrenos.

❓ Posso consultar meu projeto sem protocolo?
▼

Sim, pelo email.

Você pode usar o link do email recebido. Ou contatar o suporte com email + dados básicos para recuperar o protocolo.

❓ Os dados que envio são seguros?
▼

Sim, totalmente.

Sistema usa:

  • Conexão HTTPS (criptografada)
  • Tokens UUID únicos por projeto
  • Banco de dados PostgreSQL seguro
  • Soft delete (dados nunca são realmente apagados, apenas marcados)

Só você e revisores autenticados podem acessar seus dados.

🔧 Troubleshooting Comum
⚙️ Não recebi o email de confirmação
▼

Verifique:

  • Spam/lixo eletrônico (procure por "nexiflow" ou "noreply@...")
  • Email digitado corretamente no formulário
  • Aguarde alguns minutos (sistema pode estar processando)
  • Contate suporte com seu email registrado
⚙️ Meu link está expirado ou não funciona
▼

Links não expiram, mas verifique:

  • Copie o link inteiro (deve conter "?token=...")
  • Verifique se tem espaços extras no início/fim
  • Abra em navegador privado (incognito)
  • Contate suporte com seu protocolo
⚙️ Campo marcado como erro, mas parece correto
▼

O revisor pode ter deixado uma observação:

  • Hover sobre o ícone de erro para ver comentário
  • Revise se dados são compatíveis (ex: PIB com população)
  • Reenvie com clarificação se discordar
